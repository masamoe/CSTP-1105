JavaScript Functions 1

  1. Hello World
    Create a simple function that console.logs "Hello World". Without using a loop, invoke/call this function 3 times.
  2. Goodnight World
    Create a simple function that console.logs "Goodnight World". This time use a for loop to call the function ten times. Do not put the loop inside the function.
  3. Name and Address
    Create 2 functions: one that creates a console.log saying "My name is ..." and the other a console.log that says "I live in ...". Call both of these functions with the name function happening first.
  4. Name and Address Called Differently
    Do the same again however this time call the address function inside the name function and do it in such a way that the address log comes first. Remember you can call a function inside of another function.
  5. Choose Which Function to Call
    Create 2 functions:
    one that logs "My favourite fruit is ..." and then logs "It is grown in ...".
    The second function should log "My favourite music is ..." and then logs "... plays this kind of music".

    Then use a confirm box to only call one of these functions (Fruit or Music) based on the user's reaction e.g. ("OK = fruit, Cancel = music").
  6. Name Analysis
    Create a single function that does the following. Ask the user to type their name into a prompt. Then ask the user how old they are using a prompt. Check to see how many characters are in the name. If the name has more than 10 characters consider the name as being long. In a single console.log print the following message

    "Hello X how are you today?
    You have a long/short name which has X characters
    You are X years old and so were born in the year XXXX"

    Call this function once.
