Introduction to Programming Assignment 2

Complete the following tasks. Name your html file Name_Assignment_2 and submit it before the deadline.
60% of the marks will go to successfully completing the tasks free of errors.
20% of the marks will go to how well you comment your code. This also incorporates good naming conventions for variables & proper formatting/organisation of your code.
20% of the marks will go to how efficient your code is using the methods you have learned up to this point.

  1. Colours of the Rainbow
    • Create an array containing the seven colours of the rainbow.
    • Ask the user using a prompt what their favourite colour is.
    • Check to see if their favourite colour is in the array (casing of letters should not be relevant e.g. "Red" should equate to "red".
    • If their favourite colour happens to be one of the colours in the rainbow array, remove this colour from the array. In a single console.log print out the remaining colours separated by " *** " in reverse alphabetical order (Z -> A).
    • If their favourite colour is not in the array then print out the entire array in a single console.log separated by " xxx " in alphabetical order (A -> Z).
  2. 99 Bottles of Beer on the Wall
    Starting from 99 and working your way down to 0, print a separate console.log for each number.
    • If the number is divisible by 6, print "smash" instead of the number.
    • If the number is divisible by 4, print "bang" instead of the number.
    • If the number is divisible by both 6 and 4, print "wallop".
    The first ten console.logs should look like
    99 98 97 Wallop 95 94 93 Bang 91 Smash.

    You may want to look up the modulous operator % in JavaScript to help you figure out the divisible part of the code. Saying a number is divisible by 6 means that if I divide that number by 6 I should be left with 0 remainder. Such numbers include 6, 12, 18, 24 etc.

  3. Student Names
    The below array contains the names of each student. Do not modify the names in the array. In a single console.log state the following:
    1. There are X number of students in the array.
    2. The longest name in the array is X.
    3. There are X names that contain the letter "m" either in lower or uppercase.
    4. The following names begin with the letter "J": X, X, X, X.

    Some additional information to help you complete the tasks are as follows:

    1. ...
    2. You can assume that there is only 1 longest name i.e. no joint longest names. Also hyphens - count as characters.
    3. You should count for both lowercase "m" and uppercase "M".
    4. You need to store the names that begin with "J" in a new array and then use this array to output the message. Check for both lowercase "j" and uppercase "J" in the first letter position.
